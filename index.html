Propiedad position

Ideas principales

    La propiedad position específica el tipo de método de posicionamiento utilizado para un elemento.

    La propiedad position tiene cinco valores de posición;static,relative,fixed,absolute y sticky.

    Cada tipo de posición usa las propiedades top, bottom,left y right.

    La propiedad position static, es un tipo de posición ya predeterminado, este funciona mediante el flujo normal de la página.

    La propiedad position relative, funciona igual que el primero pero esta permite desplazar el elemento con las propiedades a partir de la posición original.

    La propiedad position absolute, funciona con el elemento contenedor, el despalzamiento se hara desde ahi.

    La propiedad position fixed, este funciona con relación a la ventana gráfica (el navegador).

    La propiedad position sticky se posiciona en funcion de la posición del desplazamiento del usuario.

Apuntes

Un elemento position static no esta posicionado de ninguna manera especial;siempre se posiciona de acuerdo con el flujo normal de la página.

El desplazamiento de la caja del posicionamiento relativo (relative) se controla con las propiedades top, bottom, left, right y z-index. Los demás elementos no ocuparan el espacio que deja el elemento desplazado.

Cuando un elemento se posicionada de forma absoluta, el resto de elementos de la página se ven afectados y modifican su posicion.

Un elemento con position fixed simpre permanece en el mismo, incluso si la página se desplaza.

Resumen

La propiedad position es una propiedad de CSS que permite posicionar los elementos adecuadamente en un proyecto web. Esta propiedad tiene cinco valores de posicion:static,relative,fixed,absolute y sticky, position static ya esta predeterminado por el lenguaje, por lo que es necesario cambiar a cualquiera de
los otros para posicionar algún elemento de otro modo. Cada valor de posición funciona de forma diferente en cada elemento.

Propiedad display

Ideas principales

    La propiedad display establece el tipo de caja que genera cada elemento.

    La propiedad display es una de las más complejas de CSS 2.1

    Cada elemento html tiene un valor de visualización predeterminado según el tipo de elemento que sea.

    El valor de visualizacion predeterminado para la mayoría de los elementos es block o inline.

    Un elemento con display block, comienza en una nueva línea y ocupa todo el ancho disponible.

    Un elemento con display inline, no comienza en una nueva línea y solo ocupa el ancho que sea necesario.

    También existe la propiedad display inline-block, este permite establecer un ancho y una altura en el elemento.

    Y por último también esta el display none, este hace que el elemento no genere ninguna caja.
    El elemento desaparece por completo del navegdor visualmente pero sigue estando en el proyecto web.

Apuntes

Los elementos con display block son los siguientes: div,h1-h6,p,form,header,footer y section.

Los elementos con display inline son los siguientes:span,a y img.

La propiedad display inline-block respeta los margin y padding, al contrario de inline que no son respetados. Tampoco agrega un salto de línea despues del elemento, por lo que el elemento puede aliniarse a otros elementos.

Si se utiliza un display none sobre un elemento todos sus descendientes también desaparecen por completo de la página.

Resumen

La propiedad display es la responsable de controlar cómo se ven los tipos de cajas que genera cada elemento y de cómo interactúan entre ellas, podiendo cambiar su apariencia modificando el valor. Cada elemento que tiene display tiene un valor de visualización predeterminado
que seria block, inline, inline-block y none.

Nuevas propiedades de CSS3

Ideas principales

    La nuevas propiedades de CSS3 buscan reducir el uso necesario de Javascript.

    Las propiedades de CSS3 son las siguientes:

    Box-radius:Genera esquinas redondeadas para la caja formada por un elemento.

    Box-shadow:Crea sombras para la caja formada por el elemento.

    @font-face:Esta regla nos permite cargar y usar cualquier fuente que necesitemos.

    Linear-gradient:Se aplica en background o background-image para agregar un gradiente lineal.

    Radial-gradient::Se aplica en background o background-image para agregar un gradiente radial.

    RGBA o HSLA:Son una de las mejoras de RGB y HSL con un canal alfa que específica la opacidad del color y saturacion

    Outline:Genera un borde alejado de un borde original del elemento.

    Border-image:Crea un borde con una imagen personalizada.

    Transform:Modifica la forma de un elemento.

Apuntes

Para algunas propiedades de CSS3 se deben usar los prefijos -moz-, -o- y -webkit- para que funcionen
en navegadores basados en motores Gecko y WebKit, como Firefox, Safari y Google Chrome

En border radius, el primer parámetro determina la curvatura horizontal y el segundo la vertical, otorgando la posibilidad de crear una elipse.

Un border-image necesita ser declarado previamiente con las propiedades border o border-with, y toma al menos tres parámetros:
La url de la imagen, el tamaño de las plezas que seran tomadas de la imagen para construir el borde y una palabra clave que específica como estas plezas serán ubicados alrededor del elemento.

Resumen

El CSS3 tiene propiedades que cubren el diseño,estilos web, estilos de movimiento de Javascript con tal de reducir su uso. Las propiedades son:box-radius, @font-face,box-shadow,linear-gradient,radial-gradient,RGBA, HSLA,outline,Border-image y transform, cada una de ellas tiene diferentes funciones y permite dar una mejor apariencia al proyecto web sin tener tanto conocimiento de Javascript.

Animación

Ideas principales

    CSS tiene permitido animar la mayoría de los elementos de Html sin usar Javascript.

    Cuando específica estilos CSS dentro de la regla @keyframes, la animación cambiara del estilo actual al estilo nuevo.

    Se utiliza animation duration para especificar cuanto tiempo debe transcurrir una animación para completar el ciclo.

    Se utiliza el animation-iteration-count para especificar el numero de veces que se debe reproducir una animación.

    Se utiliza el animation-delay para especificar un retraso para el inicio de una animación.

    Se utiliza el animation-play-state para especificar si la animación se esta ejecutando o esta en pausa.

    Se utiliza el animation timing-function para especifica la curva de velocidad de la animación (linear,ease, ease-in, ease-out,ease-in-out y cubic-bezier).

    Se utiliza el animation direction para especificar si una animación debe reproducirse hacia adelante, hacia atras o en ciclos alternos.

    Se utiliza el animation fill-mode para especificar un estilo para el elemento cuando la animación esta en pausa.

Apuntes

Para que una animación funcione se debe vincular a un elemento de la siguiente forma:
div {
animation-name:animo1;
}
@keyframes animo1 {
from {background-color:red;}
to {background-color:yellow;}
}

La duración de una animación esta predeterminado con 0s por lo que se tiene que cambiar la duración para que sea una animación real.

Se utiliza animation, una propiedad para configurar todas las propiedades de la animación.

Resumen

La animación es otra de la propiedades de CSS3 que da un estilo de movimiento a los elementos. Para agregar una animación se utiliza la regla @keyframe y con esta regla se debe de especificar el nombre de la animación, la duración, el número de veces que debe mostrarse la animación, si la animación esta en pausa o no,que curva de velocidad tiene,si va a reproducirse en hacia adelante o hacia atras, entre otros.

Referencias avanzadas con CSS3

Ideas principales

    CSS3 tiene referencias avanzadas en los atributos y elementos de Html.

    El primero es el atributo id, que es el nombre que identifica al elemento.

    El otro atributo es el class, este es mas flexible y puede ser asignado a varios elementos del Html en el documento que comparten un diseño similar.

    En cuanto referencias avanzadas en los elementos, existe el selector de elementos, este es una regla que selecciona todos los elementos de un tipo especifico en el documento Html.

    También estan las pseudoclases: nth-child(),nth-child(odd):nth-child(even),first-child,last-child, only-child y not().

    También hay referencias avanzadas en el selector atributo que este selecciona elementos Html en funcion de la presencia o el valor de uno de sus atributos.

    Y por último esta el selector de relación, que permite seleccionar elementos basados en su relación con otros elementos dentro del documento.

Apuntes

El pseudoclase nth-child() nos permite encontrar un hijo especifico.

El pseudoclase nth-child(odd):nth-child(even) se utilizan para seleccionar elementos con base en su posición en la secuencia de sus hijos, para alternar entre elementos impares y pares.

Los pseudoclases first-child y last-child seleccionan al primer y ultimo hijo.

El pseudoclase only-child afecta a un elemento siempre y cuando sea el único hijo disponible.

Resumen

CSS3 también tiene referencias avanzadas en los atributos y elementos de Html que se utilizan para delimitar los elementos Html de nuestra página web a los que queremos aplicar estilo.Estas referencias son selectores de atributos y elementos, asi como también de relación.
